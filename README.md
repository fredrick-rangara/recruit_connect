# RecruitConnect

A premium job board and recruitment platform designed to connect top talent with industry-leading employers.

## Features

- **Job Seekers**:
  - Advanced Job Search & Filtering
  - Real-time Application Tracking
  - Profile Management
  - Animated Dashboards

- **Employers**:
  - Job Posting & Management
  - ATS (Applicant Tracking System) Pipeline
  - Company Dashboard
  - Candidate Evaluation

## Tech Stack

- **Frontend**: React, Redux Toolkit, React Router, CSS3 (Premium Styles)
- **Backend**: Flask (Python)

## Getting Started

1. `cd client`
2. `npm install`
3. `npm run dev`
